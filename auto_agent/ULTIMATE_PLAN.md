## 📋 REJA: QADAMMA-QADAM

### 1️⃣ STEP 1: VISION SYSTEM (Ko'rish)
**Maqsad:** Agent screenshotlarni ko'ra oladi, ekrandagi elementlarni taniydi

**Imkoniyatlar:**
- Screenshot ni GPT-4 Vision ga yuborish
- Ekrandagi button, text, image larni taniydi
- UI elementlar bilan avtonom ishlay oladi
- "Bu nima?" deb so'rash mumkin

**Texnologiya:**
- OpenAI GPT-4 Vision API (eng kuchli)
- Yoki: Llava (offline, bepul)

**Misol:**
- "Ekrani qanday ko'rinishda?" → Screenshot oladi, tasvirlaydi
- "Bu button ni bos" → Screenshot oladi, button ni topadi, bosadi

**Kutubxonalar:**
```
openai>=1.3.0  # Vision API uchun
# yoki
llava>=1.0  # Offline vision
```

### 2️⃣ STEP 2: VOICE SYSTEM (Ovoz)

### 3️⃣ STEP 3: SELF-LEARNING ENGINE (Mustaqil O'rganish)
**Maqsad:** Agent o'z tajribasidan o'rganadi, doimiy ravishda akulli bo'lib boradi

**Imkoniyatlar:**
- **Learn from mistakes:** Xatolaridan o'rganadi
- **Knowledge base:** Foydali ma'lumotlarni saqlaydi
- **Pattern recognition:** Takrorlanuvchi vazifalarni o'rganadi
- **Self-improvement:** O'z kodini yangilaydi

**Mexanizm:**
```
1. Har bir vazifa natijasini tahlil qiladi
2. Muvaffaqiyatsiz → sababni topadi, yod oladi
3. Muvaffaqiyatli → yaxshi pattern ni saqlaydi
4. Vaqt o'tishi bilan ko'proq narsa bilib boradi
```

**Fayl tuzilishi:**
```
knowledge/
├── errors.json      # Xatolar va ularning yechimlari
├── patterns.json    # Muvaffaqiyatli patternlar
├── preferences.json # Foydalanuvchi afzalliklari
└── skills.json      # O'rganilgan ko'nikmalar
```

**Misol:**
- Birinchi marta "Excel yarat" deganingizda xato bo'ladi
- Keyingi safar xuddi shu xatoni qilmaydi
- Vaqt o'tishi bilan "Excel" bilan ishlashni o'rganadi

### 4️⃣ STEP 4: AUTONOMOUS CODE EXECUTION
**Maqsad:** Agent mustaqil kod yozadi, ishga tushiradi, xatolardan o'rganadi

**Imkoniyatlar:**
- **Auto-coding:** To'liq loyiha yaratish
- **Self-debugging:** Xatolarni o'zi tuzatadi
- **Code review:** O'z kodini tekshiradi
- **Learning:** Har bir xatodan o'rganadi

**Ishlash tartibi:**
```
1. Foydalanuvchi: "Web sayt yarat"
2. Agent: Rejalashtiradi (HTML + CSS + JS)
3. Agent: Kod yozadi
4. Agent: Ishga tushiradi
5. Natija: Muvaffaqiyatli yoki xato
6. Xato bo'lsa → O'zi tuzatadi
7. Keyingi safar yaxshiroq qiladi
```

**Maxsus funksiyalar:**
- Virtual environment yaratish
- Birnechta fayllardan tuzilgan loyihalar
- Kodni tahlil qilish va optimallashtirish
